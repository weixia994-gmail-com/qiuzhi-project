# Qiuzhi Restaurant

## Description
A management skill for restaurant operations, menu planning, and inventory tracking.

## Usage
Use this skill when the user wants to:
- Create or update a menu.
- Manage inventory lists.
- Simulate customer orders.

## Tools
- `read/write`: Manage menu and inventory files (JSON/Markdown).

## Scripts
- `scripts/menu_manager.py` (To be implemented): Add/Remove items from menu.
