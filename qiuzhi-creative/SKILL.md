# Qiuzhi Creative

## Description
A creative assistant skill for generating ideas, structuring content, and managing creative assets. Inspired by the "Qiuzhi" workflow.

## Usage
Use this skill when the user wants to:
- Brainstorm ideas for articles, videos, or projects.
- Structure a creative brief.
- Organize references and assets.

## Tools
- `read`: Read reference files.
- `write`: Save drafts and outlines.
- `web_search`: Find inspiration.
