Welcome to **Qiuzhi Project**, a curated collection of advanced AI Agent Skills designed for OpenClaw. This project demonstrates how to build, structure, and deploy powerful skills that turn your AI into a proactive employee.

### Included Skills

#### 1. [Qiuzhi Creative](./qiuzhi-creative)
**The "Brain" of the operation.**
- **Automatic Summarization**: Summarize YouTube videos and articles into structured notes.
- **Idea Management**: Capture fleeting thoughts from Telegram.
- **Content Generation**: Turn research into content drafts.

#### 2. [Qiuzhi Restaurant](./qiuzhi-restaurant)
**A simulation of complex task management.**
- **Menu Management**: Dynamic menu updates via file I/O.
- **Order Simulation**: Simulate customer workflows.
- **Inventory Tracking**: Auto-deduct ingredients based on orders.
