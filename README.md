# Qiuzhi Project

A collection of OpenClaw Agent Skills, inspired by the "Qiuzhi" Agent Skills Guide.

## Sub-Projects

### 1. [Qiuzhi Creative](./qiuzhi-creative)
Focuses on creative workflows, brainstorming, and content structuring.

### 2. [Qiuzhi Restaurant](./qiuzhi-restaurant)
Focuses on restaurant management simulation, menus, and inventory.

## Usage
Clone this repository into your OpenClaw `skills/` directory or use `sessions_spawn` to load specific skills.
